GitHub Pages Hosting

This portfolio is hosted for free on GitHub Pages. Here's how you can deploy your own portfolio:

How to Deploy This Portfolio on GitHub Pages

  1. Save this file as index.html on your computer
  2. Create a GitHub repository named yourusername.github.io (replace "yourusername" with your GitHub username)
  3. Upload the index.html file to your repository
  4. Go to repository Settings → Pages → Select "main" branch → Click "Save"
  5. Your site will be live at: https://yourusername.github.io
GitHub Pages Guide Create Repository

Custom Domain (Optional)

You can also use a custom domain with GitHub Pages:

  1. Buy a domain from a registrar (Namecheap, GoDaddy, etc.)
  2. In your repository Settings → Pages → Custom domain, enter your domain
  3. Configure DNS records with your domain provider

Free Alternative: You can use Freenom for free domain names (like .tk, .ml, .ga, .cf, .gq).
